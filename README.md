# musescoredl.github.io
Musescore downloader, allowing to skip the paywall

## How to use
### Option 1.
 - enter [musescoredl.vercel.app](https://musescoredl.vercel.app/)
 - paste the URL from Musescore
### Option 2.
 In the URL from Musescore, like `https://musescore.com/user/12345/scores/6789`, replace `musescore.com` with `musescoredl.vercel.app`, so it becomes `https://musescoredl.vercel.app/user/12345/scores/6789`.
